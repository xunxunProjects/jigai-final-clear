# 医学刷题 · 题库练习

一个轻量的医学题库练习网站，覆盖**解剖**、**组织学**、**生理**三个科目，共 670 道单选题与判断题。答题即时反馈、附带解析，进度自动保存在本地浏览器。

使用 **Vite + React + TypeScript** 构建，界面采用 Claude 风格的暖色纸感设计，支持移动端与桌面端，内置浅色 / 深色主题。

## 功能

- **三科题库**：解剖（156）、组织学（144）、生理（370），自动从 `assets/*.md` 解析。
- **即时批改**：选择后立即标出正误，高亮正确答案并展开解析。
- **每日目标**：可设置每天刷题数量，首页展示今日完成度与连续学习天数。
- **温故知新**：像百词斩 / 多邻国一样重做做过的题——「今日回顾」「重做错题」「重做做过的题」均为重新作答（可跨科目），用于巩固记忆。
- **题号导航**：网格式跳转，按正确 / 错误 / 当前 / 未答状态着色，自动定位当前题。
- **结果页**：正确率、答对 / 答错 / 未答统计，错题回顾，可「重做错题 / 随机重做 / 重做全部」。
- **进度保存**：作答记录、每日目标与主题写入 `localStorage`，刷新后可继续。
- **键盘操作**（桌面）：`A`–`E` / `1`–`9` 选择，`←` `→` 上下题。
- **响应式布局**：桌面端「上一题 / 下一题」置于顶部工具栏，移动端固定在屏幕底部。
- **深浅主题**：跟随系统，可手动切换；全站使用苹果系统字体。
- **公式排版**：支持题库中的上下标记法（`Na^+^` → Na⁺，`O~2~` → O₂）。

## 题库格式

题目放在 `assets/` 下的 Markdown 文件中，构建时通过 `import.meta.glob('/assets/*.md')` 读取。每题结构如下：

```
---questionstart
1. 题干……
A. 选项一 B. 选项二 C. 选项三 D. 选项四
---questionend
---answerstart
答案：C. 选项三
解析：……
---answerend
```

判断题以 `（ ）` 结尾，答案为 `正确` / `错误`。解析器（`src/data/parse.ts`）兼容多种排版：选项可在同一行或分行、可有或无分隔符、可粘连在一起，并能避开选项文本中的拉丁字母（如 `ATP`、`DNA`、`Purkinje`）。

```
  styles.css            主题变量与全部样式
```
